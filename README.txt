# NFL-Dynasty-Simulator

Below is a list of commands that can be used within the simulation

- "free agents" opens up the free agents menu, where you can select and sign free agents to your team
- "game" begins the weekly game loop (lets you either simulate or see the play-by-play)
- "schedule" shows the user what the matchups for the specific week are
- "roster" shows the user their teams full roster, along with the players' individual ratings and contract information
- "stats" shows the stats of each player
- "cut" lets you cut any player you choose, however you must sign a replacement before beginning a game
- "teams" lets you see the rosters of every other team in the league
- "advance" advances weeks in the season
